# Vikas Chaurasia - Personal Website

This repository contains the source code for my personal website hosted on GitHub Pages: [https://vkas-chaurasia.github.io](https://vkas-chaurasia.github.io).

## Overview
This is a static website built with HTML, CSS, and JavaScript. It serves as my personal portfolio to showcase my projects, experience, and interests.

## Structure
- `index.html`: The main entry point of the website.
- `style.css`: The stylesheet defining the website's look and feel.
- `script.js`: Any interactive logic for the website.
- `StructureBeforeSemantics.pdf`: A document hosted on the site.
- `badminton.jpg`, `hiking.jpg`: Images used on the site.
